# 2.Python(DP)

DP问题关键点

1. 最优子问题
2. 状态转移公式



Sum(i)为[0,i]表示以`i`结尾的子数组最大和

OPT(i)表示[0,i]中的最大子数组和

那么，对于数组中的元素nums[i]，我们有两种选择：

1. 将其添加到Sum(i -1)之中
2. 将其作为新的起点重新开始计算

Sum(i) = max(Sum(i - 1) + nums[i], nums[i])

OPT(i) = max(OPT(i - 1),Sum(i))
